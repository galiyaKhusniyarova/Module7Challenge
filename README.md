# Module7Challenge
The task is to build a financial database and web application by using SQL, Python, and the Voil√† library to analyze the performance of a hypothetical fintech ETF.

# Technologies
- python 3.9.12
- JupyterLab
- libraries: pandas, numpy, hvplot, sqlalchemy, voila*
- MacOS 
</br> </br> * - Voila is used to deploy the Notebook as a web application.
